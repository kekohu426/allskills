如果你第一次接触 Skills，可以按下面三步快速上手。

## Step 1：选择 Skill

进入 AllSkills.cn 的 Skills 页面，按分类或搜索找到适合你的技能。

## Step 2：复制指令

打开 Skill 详情页点击“复制 Skill”，把完整指令复制到剪贴板。

## Step 3：粘贴并补充需求

在 Claude 对话框中粘贴技能指令，追加你的具体任务。例如：

```
使用这个 Skill 帮我生成一个增长分析报告，目标用户是 SaaS 创业团队。
```

## 常见误区

- **只贴一半内容**：请完整复制 Skill 指令
- **没有给明确目标**：补充你的场景、目标、限制条件
- **一次塞太多任务**：先拆成 2-3 个清晰子任务

## 进阶建议

- 把常用 Skills 保存到 Claude Projects
- 为团队建立“技能库清单”，持续迭代
- 每次生成后把有效提示词沉淀为模板
